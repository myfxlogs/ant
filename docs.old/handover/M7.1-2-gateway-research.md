## 3. alfq mdgateway 抽象方式

alfq 使用三层适配器架构：

```
mdgateway/
  adapter/
    mtapi/client.go      — 统一 MT API 客户端（PlaceOrder, FetchHistory, etc.）
    mt4/client.go         — ≤80 行薄封装
    mt5/client.go         — ≤80 行薄封装
  runner.go              — 主循环
  bar_aggregator.go      — tick→bar
  clickhouse_writer.go   — CH 持久化
  quality.go             — 质量校验
  spill_replay.go        — 容错 spool
```

关键抽象：`Gateway` 接口统一 MT4/MT5 差异，mdgateway 不直接 import mt4client/mt5client。
