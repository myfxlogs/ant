## 1. AuthStore（认证状态持久化）

```typescript
interface AuthState {
    user: User | null;
    accessToken: string | null;
    isAuthenticated: boolean;
    _hasHydrated: boolean;  // 控制启动 loading
}

export const useAuthStore = create<AuthState>()(
    persist(
        (set) => ({
            user: null,
            accessToken: null,
            isAuthenticated: false,
            _hasHydrated: false,
            setUser: (user) => set({ user, isAuthenticated: !!user }),
            setTokens: (accessToken, refreshToken, user?) => set({
                accessToken, isAuthenticated: true, _hasHydrated: true, user,
            }),
            logout: () => set({ user: null, accessToken: null, isAuthenticated: false }),
        }),
        {
            name: 'auth-storage',              // localStorage key
            storage: createJSONStorage(() => localStorage),
            partialize: (state) => ({ user, accessToken }),  // 只持久化 user + token
            onRehydrateStorage: () => (state, error) => {
                // 恢复后设置 _hasHydrated = true
                state._hasHydrated = true;
                state.isAuthenticated = !!state.user;
            },
        }
    )
);
```

### Token 生命周期（tokenLifecycle.ts）

- `ensureFreshToken()`：每次请求前检查 token 是否即将过期，提前刷新
- `refreshAccessToken()`：401 时尝试 refresh token
- `startTokenScheduler()`：页面可见时定时刷新
- 基于 `localStorage` 持久化，刷新后 token 不丢失

### 关键方法

| 方法 | 触发时机 | 作用 |
|------|----------|------|
| `setPositions(accountId, positions)` | RPC 查询结果 | 全量替换持仓 |
| `addPosition(accountId, position)` | 流 onOrder(open) | 新增持仓（含去重） |
| `updatePosition(accountId, ticket, updates)` | 流 onOrder(modify) | 更新持仓字段 |
| `removePosition(accountId, ticket)` | 流 onOrder(close) | 移除持仓 |
| `setAccountInfoById(accountId, info)` | 流 onProfit | 批量写入财务数据 |
| `touchStreamProfitAt(accountId)` | 流 onProfit | 记录最近流更新时间 |

**设计要点**：
- `positionsMap` 按 accountId 分片存储，切换 `currentAccountId` 时自动更新 `positions`
- `setAccountInfoById` 被调用时自动将 accountId 加入 `accountReceivedData`
- `lastStreamProfitAtByAccount` 用于利润更新的节流控制

### 拦截器链

#### 拦截器 1：401 安全网（最先执行）

```typescript
(next) => async (req) => {
    try {
        return await next(req);
    } catch (error) {
        if (error instanceof ConnectError && error.code === Code.Unauthenticated) {
            // 排除 auth 相关 API
            const newToken = await refreshAccessToken();
            if (newToken) {
                req.header.set('Authorization', `Bearer ${newToken}`);
                return next(req);   // 重试原始请求
            }
        }
        throw error;
    }
}
```

#### 拦截器 2：认证 + 错误处理（第二个执行）

```typescript
(next) => async (req) => {
    // 1. Token 预刷新（避免 401）
    const token = await ensureFreshToken();
    if (token && !isAuthFree) {
        req.header.set('Authorization', `Bearer ${token}`);
    }

    // 2. 语言头
    req.header.set('Accept-Language', lang);

    try {
        return await next(req);
    } catch (error) {
        // 3. 连接失败 → Modal.error
        // 4. 业务错误 → 翻译 i18n key → message.error()
        throw error;
    }
}
```

### 错误处理

- `Failed to fetch` → `Modal.error({title, content})`，单例显示
- `Code.Unimplemented`（12）→ 直抛
- `AbortError` → 直抛
- 业务错误 → `translateMaybeI18nKey()` 翻译后端 i18n key → `message.error()`，800ms 节流

## 6. QueryProvider（TanStack Query）

```typescript
export function QueryProvider({ children }) {
    const [queryClient] = useState(() =>
        new QueryClient({
            defaultOptions: {
                queries: {
                    staleTime: 30_000,        // 30s 内不重新请求
                    retry: 2,                  // 失败重试 2 次
                    refetchOnWindowFocus: false,
                },
            },
        }),
    );
    return <QueryClientProvider client={queryClient}>{children}</QueryClientProvider>;
}
```

### useRpcQuery

```typescript
export function useRpcQuery<TData, TError = Error>(
    queryKey: QueryKey,
    queryFn: () => Promise<TData>,
    options?,
) {
    return useQuery<TData, TError, TData, QueryKey>({
        queryKey,
        queryFn,
        staleTime: 30_000,
        retry: 2,
        refetchOnWindowFocus: false,
        ...options,
    });
}
```
