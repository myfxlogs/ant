## 1. 流事件 API（stream.ts）

### 1.1 四种流类型

| 方法 | 后端 RPC | 事件类型 | 重连策略 |
|------|----------|----------|----------|
| `subscribeEvents` | `StreamService.subscribeEvents` | orderUpdate, profitUpdate, accountStatus, positionSnapshot | 指数退避 + 12次 cap |
| `subscribeProfitUpdates` | `subscribeProfitUpdates` | 单纯 profit 流 | 共享流 |
| `subscribeOrderUpdates` | `subscribeOrderUpdates` | 单纯 order 流 | 共享流 |
| `subscribeUserSummary` | `subscribeUserSummary` | 用户汇总统计 | 指数退避 + 12次 cap |

### 1.2 subscribeEvents 实现

```typescript
const runStream = async (retryCount = 0) => {
    const abortController = new AbortController();
    currentAbort = abortController;

    try {
        const stream = streamClient.subscribeEvents(
            { accountIds },
            { signal: abortController.signal }
        );

        for await (const event of stream) {
            const e = event as StreamEvent;

            switch (e.payload.case) {
                case 'orderUpdate':
                    callbacks.onOrder?.(toCamelCase(e.payload.value));
                    break;
                case 'profitUpdate':
                    callbacks.onProfit?.(toCamelCase(e.payload.value));
                    break;
                case 'accountStatus':
                    callbacks.onStatus?.(toCamelCase(e.payload.value));
                    break;
                case 'positionSnapshot': {
                    const snap = toCamelCase(e.payload.value);
                    const orders = snap.positions.map(o => toCamelCase(o));
                    callbacks.onPositionSnapshot?.(snap.accountId, orders);
                    break;
                }
            }
        }

        // 流正常结束 → 指数退避重连
        const delay = Math.min(1000 * Math.pow(2, retryCount), 30000);
        setTimeout(() => runStream(retryCount + 1), delay);
    } catch (error) {
        if (isAborted) return;
        // 传输层故障累积 12 次后停止重连
        if (isLikelyStreamTransportFailure(error)) {
            transportFailStreak++;
            if (transportFailStreak >= 12) {
                callbacks.onError?.(new Error('stream transport failure cap reached'));
                return;
            }
        } else {
            callbacks.onError?.(error);
        }
        const delay = Math.min(1000 * Math.pow(2, retryCount), 30000);
        setTimeout(() => runStream(retryCount + 1), delay);
    }
};
```

### 1.3 共享流模式（subscribeShared）

用于 `subscribeProfitUpdates` 和 `subscribeOrderUpdates`：

```typescript
type SharedStreamState<T> = {
    abortController: AbortController;
    listeners: Map<string, Listener<T>>;  // 多个订阅者共享同一底层流
    started: boolean;
};

function subscribeShared<T>(store, key, start, listener) {
    let state = store.get(key);
    if (!state) {
        state = { abortController: new AbortController(), listeners: new Map(), started: false };
        store.set(key, state);
    }
    const id = Math.random().toString(36).slice(2);
    state.listeners.set(id, listener);
    startSharedStream(state, start, key, store);
    return () => {
        state.listeners.delete(id);
        if (state.listeners.size === 0) {
            state.abortController.abort();  // 无监听器时取消流
            store.delete(key);
        }
    };
}
```

## 2. ConnectProvider（连接生命周期管理）

### 2.1 连接生命周期

```
connect()
 ├─ 检查 isAuthenticated + accessToken
 ├─ 未登录 → 清除所有流订阅, 重置状态
 ├─ 已登录 →
 │   ├─ subscribeUserSummary（用户汇总流）
 │   ├─ subscribeEvents（事件流）
 │   │   ├─ onOrder → positionsMap 更新（开/关/改）
 │   │   ├─ onProfit → 节流 300ms 批量更新
 │   │   ├─ onStatus → 更新 account 连接状态
 │   │   ├─ onPositionSnapshot → 全量替换持仓
 │   │   └─ onError → 标记断开
 │   └─ 设置 isConnected=true
```

## 4. 数据适配器（dataAdapter.ts）

所有从后端接收的数据经过 `toCamelCase()` 递归转换：

```typescript
// backend snake_case → frontend camelCase
// 例如: account_id → accountId, margin_level → marginLevel
export function toCamelCase<T>(obj: unknown): T {
    if (Array.isArray(obj)) return obj.map(toCamelCase) as T;
    if (obj !== null && typeof obj === 'object') {
        return Object.fromEntries(
            Object.entries(obj).map(([key, value]) => [
                key.replace(/_([a-z])/g, (_, c) => c.toUpperCase()),
                toCamelCase(value),
            ])
        ) as T;
    }
    return obj as T;
}
```
