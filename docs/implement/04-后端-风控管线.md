# 04 · 后端风控管线（risksvc）

**涉及源码**：`backend/internal/risksvc/pipeline.go`

---

## 1. 管线架构

```
SignalRequest
 → Stage 1: Capability      → TierCheck（用户能力等级）
 → Stage 2: HardLimit       → KYC + GeoIP + Jurisdiction 硬限制
 → Stage 3: PlatformLimits  → 全平台多头/空头敞口限制
 → Stage 4: RiskEngine      → 最大持仓数 + 保证金水平
 → Stage 5: PositionSizer   → 波动率目标 / Kelly 仓位计算
 → Stage 6: BlockAllocator  → 多账户按比例分配
 → SignalResult{Allowed, Lots, Stage}
```

---

## 2. SignalPipeline

```go
type SignalPipeline struct {
    capStore  *CapabilityStore       // Stage 1
    hardLimit *HardLimitEvaluator    // Stage 2
    platform  *PlatformAggregator    // Stage 3
    limits    *PlatformLimits        // Stage 3
    engine    *Engine                // Stage 4
    sizer     PositionSizer          // Stage 5
    allocator BlockAllocator         // Stage 6
}
```

### Process 实现

```go
func (p *SignalPipeline) Process(ctx, sig *SignalRequest) *SignalResult {
    // Stage 1: Capability
    if p.capStore != nil {
        pre := p.capStore.Get(sig.UserID).TierCheck()
        if !pre.Allowed { return &SignalResult{Allowed: false, Reason: pre.Reason, Stage: "capability"} }
    }

    // Stage 2: HardLimit
    if p.hardLimit != nil {
        if err := p.hardLimit.Evaluate(ctx, &HardLimitRequest{...}); err != nil {
            return &SignalResult{Allowed: false, Reason: err.Error(), Stage: "hardlimit"}
        }
    }

    // Stage 3: PlatformLimits
    if p.platform != nil && p.limits != nil {
        exposure := p.platform.GetSnapshot()
        if exposure != nil {
            result := p.limits.Check(exposure)
            if !result.Allowed { return &SignalResult{...Stage: "platform_limits"} }
        }
    }

    // Stage 4: RiskEngine
    if p.engine != nil {
        result := p.engine.Evaluate(ctx, &CheckRequest{...})
        if !result.Passed { return &SignalResult{...Stage: "risk_engine"} }
    }

    // Stage 5: PositionSizer
    if p.sizer == nil { return &SignalResult{...Stage: "sizer"} }
    sres, err := p.sizer.Size(ctx, &SizerRequest{...})
    if err != nil || sres.Lots <= 0 { return &SignalResult{...Stage: "sizer"} }

    // Stage 6: BlockAllocation
    if p.allocator != nil && len(sig.TargetAccounts) > 0 {
        allocs := p.allocator.Allocate(ctx, sres.Lots, sig.TargetAccounts)
        return &SignalResult{Allowed: true, Lots: sres.Lots, Allocations: allocs, ...}
    }

    return &SignalResult{Allowed: true, Lots: sres.Lots, ...}
}
```

## 3. SignalRequest

```go
type SignalRequest struct {
    UserID         string       // 用户 ID
    AccountID      string       // 交易账户 ID
    Symbol         string       // canonical symbol
    Side           string       // "buy" / "sell"
    SignalStrength float64      // 0–1 信号强度

    // 市场数据
    Price          float64
    ATR            float64      // 平均真实波幅
    AnnualVol      float64      // 年化波动率
    ContractSize   float64      // 合约规模
    HoldingDays    float64      // 持仓天数
    ContractExpiry time.Time    // 合约到期日（现货=zero）

    // 账户状态
    Balance    float64
    Equity     float64
    FreeMargin float64
    Margin     float64
    Positions  int              // 当前持仓数

    // 多账户批量交易
    TargetAccounts []AllocAccount
    ClientIP       string      // GeoIP 检查
}
```

## 4. SignalResult

```go
type SignalResult struct {
    Allowed     bool               // 是否允许交易
```

## 9. PositionSizer 接口

```go
type PositionSizer interface {
    Size(ctx, req *SizerRequest) (*SizerResult, error)
}

type SizerRequest struct {
    Symbol              string
    Price, ATR, AnnualVol, ContractSize float64
    HoldingDays         float64
    AccountID           string
    Balance, Equity, FreeMargin float64
}

type SizerResult struct {
    Lots     float64
    RiskUsed float64
    Method   string  // 计算方式名称
}
```

### 实现

**VolTargetSizer**（默认使用，RiskBudgetPct=0.01）：
```
lots = (equity * riskBudgetPct) / (ATR * contractSize * holdingDays)
```
